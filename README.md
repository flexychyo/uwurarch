1) Install CachyOS Linux (Desktop version with Hyprland). I recommend this distribution because it contains many necessary packages pre-installed. You can start with a pure Arch distribution, but finding and configuring packages will take some time.

2) Install Axe Shell. To do this, log in to Hyprland, press Win+Q to open a terminal, and enter:
curl -fsSL https://raw.githubusercontent.com/Axenide/Ax-Shell/main/install.sh | bash

3) Install UwUrArch:
curl -fsSL https://raw.githubusercontent.com/flexychyo/uwurarch/main/install.fish | fish

4) reboot

5) To open the Hyprland config:
code ~/.config/hypr/hyprland.conf
